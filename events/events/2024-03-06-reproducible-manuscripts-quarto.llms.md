Quarto makes transparent and reproducible data analysis simple and fun. In a nutshell, Quarto lets you mix text written in markdown with chunks of executable code written in, e.g., R or Python. You can use it for interactive notebooks, and export to a large number of formats including HTML, PDF/LaTeX, Microsoft Word or PowerPoint, or even interactive websites. In this workshop, we cover the fundamentals, including the markdown syntax, code chunks and inline code, and the output formats available. We also go over citations, cross-referencing, and templating, which ultimately allow you to write academic papers entirely in Quarto.

  
The self-paced tutorial for this workshop can be found here: <https://lmu-osc.github.io/introduction-to-Quarto/>

The self-paced tutorials focus on using Quarto with R and RStudio, which is also used by the instructors in their research and teaching. They are, however, no experts for every possible environment, but will try their best to assist you.

**Participants**: The targeted audience are total novices or intermediate users of literate programming tools (e.g. Rmarkdown, Quarto), who program (at any level) in R, Python, or Julia, and preferentially use RStudio (although VS Code, Jupyter, Neovim or other text editor are also suitable). Ideally, participants already know of or use version control.

**Requirement**: Participants must have their own laptop with admin access. Software will be installed as part of the workshop.
