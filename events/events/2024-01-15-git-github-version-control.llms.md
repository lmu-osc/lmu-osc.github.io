Are you working within RStudio to wrangle and analyses your data?

Join this 3 hours workshop to learn how to track and manage changes in your code so you never have to save multiple copies of your script again (e.g. before deleting a piece of code or trying a new function, or just for back-up!). Version control with Git and GitHub will make your workflow cleaner and more reproducible and allow you to code (or even write manuscripts) collaboratively.

This workshop will focus on how to use Git and Github within the RStudio IDE and consists in facilitated self-paced tutorials: <https://github.com/lmu-osc/Introduction-RStudio-Git-GitHub> and <https://github.com/lmu-osc/Collaborative-RStudio-GitHub> that include the setting up and configuring of all software required.

**Participants**: The targeted audience are novices regarding version control / Git, but who use RStudio at least occasionally. No R knowledge is necessary.

**Requirement**: Participants must have their own laptop with admin access. Software will be installed as part of the workshop. No previous knowledge necessary.
